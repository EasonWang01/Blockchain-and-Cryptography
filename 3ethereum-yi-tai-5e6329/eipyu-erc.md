# EIP \( The Ethereum Improvement Proposal \)

[https://github.com/ethereum/EIPs](https://github.com/ethereum/EIPs)

為以太坊的改進提案，類似於Bitcoin的BIP，社群所有人均可發起EIP提案，其下分為四個種類

```
1.Core
2.Networking
3.Interface
4.ERC
```
